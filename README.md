This project is a Breast Cancer Detection web application that predicts whether a tumor is Benign (B) or Malignant (M).
The web app is built with HTML for the frontend, Python Flask for the backend, and uses a scikit-learn machine learning model (model.pkl) for prediction.
Users can enter tumor features as comma-separated numbers, which are then processed by the backend to generate a prediction. 
The dataset used for training the model includes the labels B for benign (healthy) tumors and M for malignant (cancerous) tumors.
Benign tumors are not cancerous and usually less dangerous, while malignant tumors require medical attention.
